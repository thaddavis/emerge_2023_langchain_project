# Messing with langchain

## Up and running

```.sh
poetry init
poetry shell
```

## Run scripts defined in the pyproject.toml like so

```.sh
poetry run main
```

## Will need to setup OpenAI API key

```.sh
export OPENAI_API_KEY="..."
```

## Will need to setup SerAPI key for examples 4, etc.

```.sh
export SERPAPI_API_KEY="..."
```
